DataViz in ggplot2

This document provides a brief introduction to, and series of examples of, using ggplot2 with the Urban Institute ggplot2 theme created by Ben Chartoff with contributions from Aaron Williams. Before you continue, you should follow the link above to the GitHub page for the ggplot2 theme and read the instructions and caveats written there.

The tabs on the left side of this page will take you to a guide to creating various graph types using ggplot2.
